# 같이보기 World Cup Watch Party

각자가 자기 브라우저에서 치지직 같은 공식 중계를 열고, 그 화면을 자기 방 화면에
띄운 채 친구들과 실시간 채팅하는 워치파티 앱입니다.

이 앱은 다른 사람에게 영상을 재송출하지 않습니다. 각 사용자가 `내 화면 띄우기`를
눌러 자기 치지직 탭/창/화면을 선택하면, 선택한 화면이 자기 브라우저 안의 큰
플레이어에만 표시됩니다. 채팅, 참여자 목록, 준비 상태만 방 전체에 공유됩니다.

## 실행

```bash
npm install
cp .env.example .env
npm run dev
```

`http://localhost:3000`에서 새 방을 만들고 생성된 `/r/{방코드}` 링크를
공유합니다.

## 사용 방식

1. 방장이 방을 만들고 링크를 공유합니다.
2. 참가자는 닉네임으로 입장합니다.
3. 모두 `치지직에서 보기`를 눌러 공식 중계 페이지를 엽니다.
4. 방 화면에서 `내 화면 띄우기`를 누릅니다.
5. 브라우저 화면 선택 창에서 자신의 치지직 탭이나 창을 선택합니다.
6. 선택한 치지직 화면은 자기 브라우저의 큰 화면에만 보입니다.
7. 옆 채팅창에서는 같은 방 사람들과 계속 대화합니다.
8. 준비가 되면 `시청 준비 완료`를 눌러 서로 상태를 확인합니다.

중요: 여기서 화면 공유는 “내 화면을 내가 보기 위한 로컬 미리보기”입니다. 다른
참여자에게 내 화면이 전송되지 않습니다.

## 설정

```dotenv
PORT=3000

OFFICIAL_WATCH_URL=https://chzzk.naver.com/home/sports/fifa-worldcup-2026
OFFICIAL_WATCH_LABEL=치지직에서 보기
STREAM_LABEL=2026 FIFA 월드컵

# 공식 제공자가 iframe 삽입을 허용할 때만 사용하세요.
# OFFICIAL_EMBED_URL=https://example.com/official/embed
```

- `OFFICIAL_WATCH_URL`: 참가자가 공식 중계를 새 창으로 열 주소입니다.
- `OFFICIAL_WATCH_LABEL`: 버튼에 표시할 문구입니다.
- `STREAM_LABEL`: 방 상단에 표시할 경기/중계 이름입니다.
- `OFFICIAL_EMBED_URL`: 선택 사항입니다. 공식 제공자가 iframe을 허용할 때만
  앱 안에 표시됩니다.

## 왜 이렇게 만들었나요?

치지직/공식 스포츠 중계는 DRM 또는 브라우저 보안 정책 때문에 다른 사람에게
캡처 화면을 전송하려고 하면 검은 화면이 될 수 있습니다. 또한 WebRTC P2P 송출은
참가자가 늘어날수록 송출자 브라우저 부담이 커집니다.

그래서 이 프로젝트는 다음 구조로 동작합니다.

- 영상: 각자 공식 사이트에서 직접 재생
- 화면 띄우기: 각자 자기 브라우저에서만 로컬 미리보기
- 앱 서버: 방 만들기, 초대 링크, 채팅, 참여자/준비 상태
- 서버 영상 트래픽: 없음
